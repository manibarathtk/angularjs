Prison Escape

A group of N people are locked in N cells of a prison and are looking to escape. These people can send messages to each other via a messaging device only if they know each other. For transfer of any message from receiver to sender it takes 1 unit of time. To escape prison a secret number needs to be entered in dial lock of 3 of the cells. One of the member A is somehow able to find the secret number to be entered and also the cell locations B C & D from where this number needs to be dialed. Find the minimum time it would take for them to escape.


Input Format
You will be given a function which contains following arguments -
N - no of people
A - index of person knowing secret number
B C D - array of index of person who need to receive this secret number
p1 - p1 , p2 p2 array of strings where each string represents the friend pair who know each-other.

Output Format
You need to return time T to escape from the given function if possible otherwise -1.


Sample Test Case
Sample Input
5
4
3
1
2
3
5
3 2
2 4
2 5
4 5
1 5

Sample Output
2

Explanation
Here message should start from 4 and has to reach 1, 2 and 3.
For message to reach 1 it takes 4->5, 5->1 hence time taken is 2
for 2 it takes 4->2 and time taken is 1
for 3 it takes 4->2, 2->3 and time taken is 2
so the earliest time to escape is 2.
